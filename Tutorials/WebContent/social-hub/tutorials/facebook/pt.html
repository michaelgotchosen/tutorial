Social Hub tutorial

Facebook Requirments

Para ativar o Social Hubdo Facebook, precisamos dos seguintes item:

  • O ID do Aplicativo;
  • A Chave Secreta do Aplicativo
  • O "Slug Name" ou ID da página do Facebook (que deseja exibir os posts);


Steps

Criando uma aplicação no Facebook 

Para criar uma aplicação do facebook, acesse: https://developers.facebook.com

 

 

  1. Tornando-se um desenvolvedor

               (Obs.: Se seu usuário já é um desenvolvedor, pule este tópico.)

Para criar um aplicativo no Facebook, você precisa se cadastrar como um desenvolvedor e aceitar os Termos e Policiais do Facebook.

 

 

  1. Criando o APP

 Se for sua primeira aplicação, clique sobre o botão "Criar aplicativo".Caso já tenha outras aplicações criadas, você verá uma lista de aplicativos e pode criar um novo através do botão "Adicionar um novo aplicativo"ou através do menu "Meus aplicativos -> Adicionar um novo aplicativo".

 

 

 

Defina o nome de exibição como GotChosen APP, um e-mail para contato e a categoria do APP, em seguida clique em "Criar um número de identificação do aplicativo".

 

 

Clique em "Configurações -> Básico"para acessar a tela de detalhes da aplicação.

 

 

  1. Definindo o domínio do aplicativo

No campo "Domínios do aplicativo"adicione o domínio gotchosen.com. Isso dará permissão

- aplicação do GotChosen à obter informações da API do Facebook.

 

  1. Adicionando uma plataforma

Após definir o domínio do aplicativo, clique em "Adicionar plataforma".

 

 

Selecione a plataforma "Site".

 

 

Em seguida, no campo "URL do site"define como https://gotchosen.com.

 

 

Agora podemos salvar as alterações, para isto, clique no botão "Salvar alterações",que se encontra no rodapé da página, ao lado direto.

 

 

 

5. Identificando o "Slug Name" ou ID da página

Para identificar o "Slug Name"da página é bem simples, basta acessar a mesma. O Slug Nameestará contido na URL.

 

 

Caso a URL contenha uma sequência numérica (podendo conter o nome da página ou não), isso significa que a página não possui um "Slug Name",neste caso, esta sequência numérica é o ID da mesma.
