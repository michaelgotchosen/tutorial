Facebook Requirments

To Activate your Facebook inside the Social Hub, we need the following items:

  • The App's ID
  • The App's secret Key
  • The App Name


Steps

Creating an application in facebook

To create an application in facebook you need to go to : https://developers.facebook.com

 

 

  1. Becoming a developer

               (FYI: If you are already a Facebook developer, please skip this step.)

For you to create a Facebook app you first need to become a Facebook developer and accept their terms and conditions

 

 

  1. Creating an APP

 If this is your first Facebook App please click on the "Create App" button. If you already have an App, you will see a list of your apps and a button to create a new one called "Create a new App"

 

 

Define your App name as GotChosen APP, input an email for contact and also choose a caterogry that best fits you, and click "Save".

 

 

After you will need to click on Settings -> basic to access application detail modal

 

 

  1. Defining your Application domain

Inside the App Domains field add gotchosen.com. This will give GotChosen Permisson

- GotChosen can now access Facebook's API to obtain information

 

  1. Adding a platform

After defining the App domain, click on "Add platform".

 

 

Select "Site" as your platform .

 

 

Inside the URL field type https://gotchosen.com.

 

 

Now we can save all our changes by just clicking on the "Save Changes" button on the lower right side of the page.

 

 

 

5. copy your "App ID", "App Secret" and Display Name to the Gotchosen Facebook module

Display Name = Name
App ID = Key
App Secret = Secret
