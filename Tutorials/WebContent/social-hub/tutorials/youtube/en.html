Youtube

Social Hub tutorial

Youtube Requirments

To active Youtube's Social Hub we need the following items:

  • One API Key
  • Your youtube channel ID

Steps

To create a youtube application you first need to go to: https://console.developers.google.com/apis/api/youtube/overview

  1.  Creating a new project

Once inside the YouTube Data Api control panel, click the "Create project", And after define the name of the project as "GotChosen APP"and click on "Create".

  1. Activating the project

After we've created the project we need to activate it, and to do so just click on "Activate".

  1. Creating the credentials

To create your credentials all you have to do is click on "Create credentials".

  1. Assinging your credentials to your project
  • Inside the "Where will you call this API from" field select "Web Server".
  • Inside the "Which data will you access" field select "Public Data".
  • Click on "which credentials i need" button.
  1. Limiting the key access.

Your Key was created! We recommmend to limit the access to the API through specfic domains only, so therefore it can not be access from anywhere.

 To do so, just click on the "Restrict Key" button.

Inside the credentials modal, You will need to define the name as gotchosen-key-api, inside the field "Key restriction" select Website referrers (HTTP), and lastly add .gotchosen.com/* below so our application has permisson to access the API, and to finish off just click "Save".

6. identifying your channel's ID

To identify your youtube channel's ID which you would like for us to activate inside the Social Hub is simple

All you have to do is access your youtube channel through the search and observe your URL. Sometimes it will be an ID but it can also be your "Slug" name which us wand copy your slug name which is usually your channel's name

In the case that it shows your "Slug name" instead of an ID, you will need to access a video from that channel and click on the channel's name below the video's player.

Once you access the channel through the video you will see your ID inside the URL. That is the ID we need.
