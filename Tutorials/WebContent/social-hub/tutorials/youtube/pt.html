Social Hub tutorial

Youtube Requirments

Para ativar o Social Hub do Youtube, precisamos dos seguintes item:

  • Uma chave para a API válida;
  • O ID do canal (que deseja exibir os vídeos);

Steps

Para criar uma aplicação do Youtube, acesse: https://console.developers.google.com/apis/api/youtube/overview

  1.  Criando um novo projeto

Na tela do painel do YouTube Data API v3,clique no botão "Criar projeto".Em seguida, defina o nome do projeto como "GotChosen APP".e clique em "Criar".

  1. Ativando o projeto

Depois de criar um novo Projeto, você precisa ativá-lo, para isso clique no botão "Ativar".

  1. Criando as credenciais

Para criar as credenciais necessárias para ativar o Social Hub, clique em "Criar credenciais".

  1. Adicionando credenciais ao projeto
  • No campo "De onde você chamará a API?"selecione "Servidor da Web (por exemplo, node.js, Tomcat)".
  • No campo "Que dados você acessará?" selecione "Dados públicos".
  • Clique em "Preciso de quais credenciais?".
  1. Limitando o acesso à Chave

Sua chave já foi gerada. Entretanto é altamente recomendado que você limite o acesso à API através de domínios específicos, para que não seja acessado de qualquer lugar.

 Para limitar o acesso, clique em "Restringir chave".

Na tela de "Credenciais",defina o nome como gotchosen-key-api, em seguida, no campo "Restrição de chave",selecione "Referenciadores de HTTP (sites da Web)".No campo "Referenciadores de HTTP (sites da Web)",adicione *.gotchosen.com/* para que a aplicação do GotChosen tenha permissão de acesso à API. Para finalizar, clique em "Salvar".
