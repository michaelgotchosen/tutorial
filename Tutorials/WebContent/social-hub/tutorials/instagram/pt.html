Social Hub tutorial

Instagram Requirments

  • Para ativar o Social Hub do Instagram, precisamos dos seguintes itens:
    • Client ID
    • Client Secret
    • O ID do Aplicativo
    • O "Slug Name" (que deseja exibir os posts)



    Steps

    1. Criando uma aplicação no Instagram

    Para criar uma aplicação do Instagram, acesse: https://www.instagram.com/developer

    Obs.: Se seu usuário já é um desenvolvedor, pule este tópico.

     

    Para criar um aplicativo no Instagram, você precisa se cadastrar como um desenvolvedor e aceitar os Termos e Políticas.

    2) Criando o APP

    Clique em "Registre a New Client" para criar uma nova aplicação.

    Defina o "Application Name" como GotChosen APP, "Description" como Aplicação para interação social entre o publisher e o GotChosen, "Company Name" como GotChosen, "Website URL", "Valid redirect URIs" e "Privacy Policy URL" como https://www.gotchosen.com  em seguida clique em "Register".

     

     

    3) Obtendo o "Client ID" e "Client Secret"

    Na página das aplicações, identifique a aplicação "GotChosen API" e clique em "MANAGE"
